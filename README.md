# Awing AI Learning

A lightweight on‑device AI application to teach the Awing language across Beginner, Medium, and Expert levels. Built with Flutter for Android and iOS.

## Features

- AI‑Powered interactive lessons.
- Real‑time speech recognition for pronunciation practice.
- Grammar quizzes and exercises.
- Instant feedback mechanism.

## Prerequisites

- **Flutter SDK** 3.22+ (includes Dart 3.4+)
- **Android Studio** with Android SDK (for Android builds)
- **Xcode** 15+ on macOS (for iOS builds)
- **Python** 3.10+ (for model conversion only)

Run `flutter doctor` to verify your setup.

## Getting Started (Windows 11)

```bat
REM Clone the repository
git clone https://github.com/yourorg/awing-ai-learning.git
cd awing-ai-learning

REM Install all dependencies (Flutter + Python)
scripts\install_dependencies.bat

REM Or manually install Flutter deps only
flutter pub get

REM Run the app on a connected device or emulator
flutter run
```

## Building for Android and iOS

**Android** (from Windows or any OS):

```bat
flutter build apk --release
flutter build appbundle --release
```

The APK is output to `build\app\outputs\flutter-apk\app-release.apk`.

**iOS** (requires macOS with Xcode):

```bash
flutter build ios --release
```

Then open `ios/Runner.xcworkspace` in Xcode to archive and distribute.

## One‑Click Build Script (Windows)

```bat
scripts\build_and_run.bat
```

This converts the TFLite model, builds the Android APK, and launches the app.

## Model

The AI model is a small quantized TFLite file located in `assets\model.tflite`. To regenerate it from a HuggingFace model, run:

```bat
python scripts\convert_model.py
```
