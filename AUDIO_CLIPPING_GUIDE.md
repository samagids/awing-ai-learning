# Audio Clipping Guide for Awing AI Learning

## Overview
This guide tells you exactly how to clip and name audio files from the YouTube videos
so they plug directly into the app. The app will automatically detect and use any audio
files you place in the correct folders.

## Tools You'll Need
- **Video downloader**: Use any YouTube downloader to save videos as MP4
- **Audacity** (free): https://www.audacityteam.org/ — for clipping audio
  - Open the video/audio file in Audacity
  - Select the section where a word is spoken
  - File → Export → Export as MP3
  - Save with the exact filename from the lists below

## Audio File Requirements
- **Format**: MP3 (128kbps or higher)
- **Length**: 1-3 seconds per clip (just the word, trim silence)
- **Quality**: Clear pronunciation, minimal background noise
- **Volume**: Normalize to similar levels across all clips

## Tips for Clipping

1. **Start with the alphabet** — Videos #1, #2, #5, and #8 are alphabet lessons where each letter is pronounced individually. These are easiest to clip.

2. **Listen for example words** — The alphabet videos often include example words after each letter sound. Clip those for the vocabulary files.

3. **Trim silence** — In Audacity, select the silence before/after the word and delete it. Leave about 0.1 seconds of silence on each end.

4. **Normalize volume** — In Audacity: Effect → Normalize → OK. This makes all clips the same volume.

5. **You don't need all 67 words** — The app will use TTS for any missing audio files. Start with the alphabet (31 clips) and add vocabulary clips over time.

6. **Test as you go** — After adding some clips, rebuild the APK and test on your device. The app automatically picks up new audio files.
